# DOC-FLOW
A backend system that ingests documents, processes them asynchronously, extracts structured data, and exposes workflow/status APIs.

# Table of content

## Purpose
 This project have features that can be used to summarize, extract and convert documents from one format to another

## Technologies
python, fastapi, celery, docker, postgresql
